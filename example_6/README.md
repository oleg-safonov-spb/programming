# 6. Реализация линейных списков

## Задание: Реализация однонаправленного списка (Л1)

Создать структуру Student, содержащую ФИО, номер группы и средний балл.

Реализовать набор функций для работы с однонаправленным списком студентов:
* insert_front(): добавление нового студента в начало списка.
* delete_by_name(): удаление студента по фамилии (учесть однофамильцев).
* find_best(): поиск студента с наивысшим средним баллом.
* print_list(): печать всего списка.
* clear_list(): освобождение всей динамической памяти.

Написать Makefile для сборки и тестирования. Требования:
* Стандарт C17, флаги `-Wall` и `-Werror`.
* Раздельная компиляция и линковка.
* Цели: `all`, `clean` и `test` (запуск юнит-тестов).

## Запуск

```bash
make
./student_app
```

## Тестирование

```bash
make test
```

## Проверка на утечки

```bash
valgrind --leak-check=full ./student_app
```
