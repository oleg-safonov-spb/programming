# 5. Использование регулярных выражений в программах

## Задание: Валидация IP-адресов

Написать программу на C, которая считывает строки из стандартного ввода.
Используя регулярное выражение и функцию regexec, определить, является ли каждая 
строка корректным IPv4-адресом в формате XXX.XXX.XXX.XXX.
Проверить, что каждый из четырех сегментов находится в диапазоне от 0 до 255.

Написать Makefile для сборки и тестирования. Требования:
* Стандарт C17, флаги `-Wall` и `-Werror`.
* Раздельная компиляция и линковка.
* Цели: `all`, `clean` и `test` (запуск юнит-тестов).

## Запуск

```bash
make
./ip_check_app
```

## Тестирование

```bash
make test
```
