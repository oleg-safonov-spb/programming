# 1. Управляющие конструкции

## Задание: Анализ успеваемости студентов

Написать программу на C, которая считывает оценки 1-5 (или -1 для не 
аттестации) из консоли до тех пор, пока не будет введен 0.

Для введенных оценок вычислить:
* Количество "5", "4", "3", "2", "1", "не аттестовано".
* Процент "5" и "4".
* Процент "5" и "4" и "3".

## Запуск

```bash
gcc -Wall -Werror main.c -o grade_app
./grade_app
```
