# 7. Работа с файлами и файловой системой в программах

## Задание: Бинарная сериализация данных

Использовать структуру Student из предыдущего задания 
[Реализация однонаправленного списка (Л1)](../example_6).

Написать две функции:
* save_to_file(list, filename): сохраняет данные из списка в бинарный файл 
(используя fwrite).
* load_from_file(filename): читает данные из бинарного файла и воссоздает список 
в памяти (используя fread).

Написать программу на C, которая демонстрирует сохранение и последующее 
восстановление данных.

Написать Makefile для автоматизации сборки. Он должен:
* Содержать цели all, clean.
* Компилировать каждый файл отдельно в .o, а затем линковать их.
* Использовать флаги компиляции -Wall (вывод всех предупреждений) и -Werror 
(трактовка предупреждений как ошибок).

## Запуск

```bash
make
./serialize_app
```
