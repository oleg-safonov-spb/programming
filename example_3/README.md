# 3. Указатели и массивы

## Задание: Динамическая матрица

Написать программу на C, в которой пользователь вводит размеры двумерного 
массива N * M.
Динамически выделить память под матрицу (массив указателей).
Заполнить её случайными числами.
Получить транспонированную матрицу.
Обязательно: корректно освободить память через free() в конце работы.

Написать Makefile для автоматизации сборки. Он должен:
* Содержать цели all, clean.
* Компилировать каждый файл отдельно в .o, а затем линковать их.
* Использовать флаги компиляции -Wall (вывод всех предупреждений) и -Werror 
(трактовка предупреждений как ошибок).

## Запуск

```bash
make
./matrix_app
```

## Проверка на утечки

```bash
valgrind --leak-check=full ./matrix_app
```
