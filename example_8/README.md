# 8. Реализация динамических структур данных стек и очередь

## Задание: Реализация Стека на базе однонаправленного списка

Реализовать абстрактный тип данных "Стек" (LIFO), используя однонаправленный 
список.

Реализовать функции:
* push(): добавление элемента на вершину.
* pop(): извлечение элемента с вершины (с проверкой на пустоту).
* peek(): просмотр верхнего элемента без извлечения.

Написать программу на C, которая использует этот стек для проверки баланса 
скобок в математическом выражении ({[ ]}).

Написать Makefile для сборки и тестирования. Требования:
* Стандарт C17, флаги `-Wall` и `-Werror`.
* Раздельная компиляция и линковка.
* Цели: `all`, `clean` и `test` (запуск юнит-тестов).

## Запуск

```bash
make
./bracket_app
```

## Тестирование

```bash
make test
```
