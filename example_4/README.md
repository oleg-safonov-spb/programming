# 4. Стандартная библиотека

## Задание: Массив сотрудников

Создать структуру Employee (фамилия, отдел, зарплата, стаж).
Сгенерировать массив из таких структур.

Использовать функцию qsort для реализации многоуровневой сортировки:
* Сначала по названию отдела (по алфавиту).
* Внутри отдела — по убыванию зарплаты.

Вывести результат в виде отформатированной таблицы.

Написать Makefile для сборки и тестирования. Требования:
* Стандарт C17, флаги `-Wall` и `-Werror`.
* Раздельная компиляция и линковка.
* Цели: `all`, `clean` и `test` (запуск юнит-тестов).

## Запуск

```bash
make
./employee_app
```

## Тестирование

```bash
make test
```
