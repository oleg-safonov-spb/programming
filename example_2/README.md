# 2. Сборка программ

## Задание: Геометрические вычисления

Создать проект из трех файлов:
* geometry.h: прототипы функций для расчета площади и периметра фигур (круг, 
прямоугольник, треугольник) и константа PI.
* geometry.c: реализация этих функций.
* main.c: интерфейс взаимодействия с пользователем.

Написать Makefile для автоматизации сборки. Он должен:
* Содержать цели all, clean.
* Компилировать каждый файл отдельно в .o, а затем линковать их.
* Использовать флаги компиляции -Wall (вывод всех предупреждений) и -Werror 
(трактовка предупреждений как ошибок).

## Запуск

```bash
make
./geo_app
```
