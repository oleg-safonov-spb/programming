# 2. Сборка программ

## Задание: Геометрические вычисления

Создать проект из файлов:
* geometry.h: прототипы функций для расчета площади и периметра фигур (круг, 
прямоугольник, треугольник) и константа PI.
* geometry.c: реализация этих функций.
* main.c: интерфейс взаимодействия с пользователем.
* tests.c: юнит-тесты с использованием assert.

Написать Makefile для сборки и тестирования. Требования:
* Стандарт C17, флаги `-Wall` и `-Werror`.
* Раздельная компиляция и линковка.
* Цели: `all`, `clean` и `test` (запуск юнит-тестов).

## Запуск

```bash
make
./geo_app
```

## Тестирование

```bash
make test
```
